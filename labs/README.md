# Labs - Instructor Walkthroughs

This folder holds instructor walkthrough notes for the six chapters of the GH-600 course. Each chapter folder pairs with one or two Certy lessons and a hands-on practice repository under the [CertyPro organisation](https://github.com/CertyPro).

The notes are written for an instructor (or a self-directed learner playing both roles): they give the teaching goal, the running order, the key points to land, common mistakes, and how the chapter maps to the exam.

| Chapter | Topic | Domain | Weight | Practice repo |
| --- | --- | --- | --- | --- |
| [Chapter 01](./chapter-01-agent-architecture) | Agent Architecture, Planning & Validation | 1.0 | 15% | [agent-starter](https://github.com/CertyPro/certy-gh600-agent-starter) |
| [Chapter 02](./chapter-02-tooling-mcp) | Tooling, MCP Servers & Execution Context | 2.0 | 25% | [mcp-server-lab](https://github.com/CertyPro/certy-gh600-mcp-server-lab) |
| [Chapter 03](./chapter-03-memory-state-github-context) | Memory, State & GitHub Context | 3.0 | 25% | [memory-state-lab](https://github.com/CertyPro/certy-gh600-memory-state-lab), [github-control-plane-lab](https://github.com/CertyPro/certy-gh600-github-control-plane-lab) |
| [Chapter 04](./chapter-04-evaluation-debugging) | Evaluation, Debugging & Iteration | 4.0 | 15% | [evaluation-debugging-lab](https://github.com/CertyPro/certy-gh600-evaluation-debugging-lab) |
| [Chapter 05](./chapter-05-multi-agent-orchestration) | Multi-Agent Systems & Orchestration | 5.0 | 10% | [multi-agent-orchestration-lab](https://github.com/CertyPro/certy-gh600-multi-agent-orchestration-lab) |
| [Chapter 06](./chapter-06-governance-guardrails) | Governance, Guardrails & Compliance | 6.0 | 10% | [governance-guardrails-lab](https://github.com/CertyPro/certy-gh600-governance-guardrails-lab) |

Work through the chapters in order. Chapter 01 sets up the agent-starter you extend throughout, and the [capstone](../challenge-project) ties all six together.
